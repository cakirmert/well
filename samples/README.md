# Email Samples

Place redacted booking emails here as `.eml` files.

Useful examples:

- `booking-confirmation-1.eml`
- `booking-confirmation-2.eml`
- `booking-reminder-1.eml`
- `cancellation-1.eml`

Redact personal data if you want, but keep the structure, subject, sender, date/time text, studio name, and cancellation wording intact. The parser needs those fields.
